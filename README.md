# AutoCredit

AutoCredit adalah aplikasi simulasi kredit kendaraan berbasis Laravel yang membantu pengguna menghitung estimasi cicilan mobil secara cepat dan transparan.

---

##  Fitur Utama

-  Simulasi cicilan mobil
-  Perhitungan DP, bunga, dan tenor
-  Form kontak (email real)
-  UI modern (Tailwind CSS)
-  Interaksi smooth (AJAX & animation)
